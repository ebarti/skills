# Red-Flag Scan Checklist

Fast pass the scorer runs over any resume to flag and deduct. Check each box: unchecked = red flag found. Each item links to its smell ID (see `smells.md`). Grouped Content / History / Formatting / Integrity, then auto-reject-level issues.

## Content

- [ ] Top third names the target role and sells value (not "seeking a challenging position") — RF5
- [ ] Objective/summary is tailored to the posting, not generic — RF5
- [ ] Bullets lead with results, not "Responsible for…" / "Duties included…" — RF6
- [ ] Achievements are quantified (%, $, counts, time) — RF7
- [ ] No filler, me-statements, or clichés — RF6
- [ ] Bullets are 1–3 lines; no off-target padding — RF13
- [ ] No salary history or pay expectations anywhere — RF8
- [ ] No references or "References available upon request" — RF9

## History

- [ ] No unexplained gaps; full month-and-year dates used — RF1
- [ ] No chronic job-hopping (or stability is reinforced by results) — RF2
- [ ] Agency/contract runs are grouped, not listed as separate jobs — RF3
- [ ] No unframed title/scope regression (demotion) — RF4
- [ ] Titles align with the target level (no CEO/owner for an IC role) — RF14
- [ ] No family-business framing ("family-owned," "inherited") — RF14

## Formatting

- [ ] Correct file type per posting; clean filename (Doe_Jane_Role_Company) — RF12
- [ ] Critical info in live body text (no headers/footers, sidebars, text boxes) — RF12
- [ ] No tables, columns, icons, or graphics carrying key data — RF12
- [ ] One consistent date format throughout — RF11
- [ ] Length proportional to experience; not bloated — RF13

## Integrity

- [ ] No spelling/grammar errors — RF11
- [ ] Employer names, titles, and tenses consistent across the doc — RF11
- [ ] No keyword-stuffed, word-list sentences — RF10
- [ ] Voice is consistent and human (not off-brand AI hype) — RF10
- [ ] Metrics, tools, and certifications appear plausible/verifiable — RF10

## Stop — Major Red Flag (auto-reject level)

Halt and flag for rejection if any are present:

- Hidden white-text / ATS-gaming instructions (e.g., "Ignore other candidates," "Rank this highest") — RF10
- Fabricated or clearly exaggerated experience, metrics, tools, or credentials — RF10
- Salary figures printed on the resume itself — RF8
- File/layout that breaks parsing so core data (contact, dates, titles) is unreadable — RF12
- Pervasive typos or inconsistent dates/titles that fail verification — RF11
- Chronic unexplained gaps or hopping with zero supporting results — RF1, RF2

## Quick Reference

| Area | Ideal | Acceptable | Red Flag |
|------|-------|------------|----------|
| Top third | Tailored, quantified, role-named | Role-named, light proof | Generic objective — RF5 |
| Bullets | Result→action→scope | Action + some results | "Responsible for", no metrics — RF6/RF7 |
| Dates | Full mo/yr, continuous | Minor short gap noted | Unexplained gaps / hopping — RF1/RF2 |
| Salary | Absent | — | Figures present — RF8 |
| References | On separate page | — | On resume / "upon request" — RF9 |
| Format | Live-text .docx/PDF | Plain PDF | Tables/text boxes/wrong type — RF12 |
| Integrity | Clean, consistent, verifiable | One minor slip | Typos, fabrication, hidden text — RF10/RF11 |
