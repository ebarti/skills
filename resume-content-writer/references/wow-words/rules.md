# Wow Words Rules

Rules for using power language so every resume bullet leads with strength and substance.

## Core Rules

### 1. Start every bullet with a strong verb

Open each line with an action verb, not a pronoun, article, or filler.

- Drop first- and third-person pronouns (no "I", "he/she", "they").
- Skip most articles ("a", "an", "the") unless they clarify meaning.
- Resume language isn't conversational — keep it tight and professional.

**Bad → Good**:
- `Responsible for the team's onboarding` → `Onboarded 40+ new hires across two regions`

### 2. Use consistent past tense

Report completed results, so write job descriptions in past tense — even for your current role.

- Current-role bullets still use past tense (you're reporting completed results).
- Don't mix tenses within or across bullets.

**Formula**: Active verbs + consistent past tense + short, clear lines = winning content.

### 3. Avoid weak, passive openers

Never lead with limp phrases that describe duties instead of results.

Banned openers:
- "Responsible for ..."
- "Helped with ..."
- "Worked on ..."
- "Duties included ..."
- "Assisted in ..." (when a stronger, accurate verb exists)

These flatten the bullet and bury the outcome. Replace with a precise action verb.

### 4. Don't repeat the same verb

Avoid reusing verbs; strive for original use for each bullet.

- Scan the whole resume for repeated verbs and vary them.
- If two bullets both start with "Managed", upgrade one (e.g., "Directed", "Oversaw", "Led").

### 5. Match the verb to the target function

Pick a precise verb from the functional category that fits the target job.

- Use the nine categories (see `examples.md` / `knowledge.md`) to choose.
- Mirror the language of the target job description.

### 6. Lead with the outcome (outcome-first)

Order each bullet: result first, then the action and challenge.

- Start with a result, pick a precise verb, then anchor it with keyword nouns.
- This reorders a CAR (Challenge–Action–Result) story to put the payoff up front.

### 7. Avoid empty buzzwords and clichés

Choose words of power and substance — not filler that says nothing.

- Skip vague hype with no evidence behind it.
- Swap insider jargon for language used in the target job description (then verify accuracy).
- Keep verbs authentic to what you actually did.

## Guidelines

Softer recommendations:

- Pair every action verb with keyword nouns so you pass the scan *and* impress the human.
- Split long sentences into short, clear lines.
- When using AI to upgrade verbs, keep the same facts — no embellishment — and verify before adding.
- Triple-check spelling visually; don't rely on spell-check alone (one misspelling can screen you out).

## Exceptions

When these rules may be relaxed:

- **Articles for clarity**: Keep an article when dropping it changes meaning.
- **Present tense**: Only the chapter's rule stands — use past tense even for current roles; there is no general present-tense exception for completed results.

## Quick Reference: Weak → Strong Swaps

| Weak / passive opener | Strong replacement (pick by function) |
|-----------------------|----------------------------------------|
| Responsible for | Directed, Managed, Led, Oversaw |
| Helped with | Supported, Facilitated, Drove, Enabled |
| Worked on | Built, Developed, Engineered, Produced |
| Duties included | Delivered, Executed, Coordinated |
| Assisted in | Partnered to, Contributed to, Supported |
| Was in charge of | Headed, Spearheaded, Owned |
| Made / did | Created, Produced, Achieved |

## Quick Reference: Rules at a Glance

| Rule | Summary |
|------|---------|
| Strong verb first | Open every bullet with an action verb |
