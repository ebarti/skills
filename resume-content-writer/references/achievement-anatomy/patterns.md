# Achievement Anatomy Patterns

Fill-in-the-blank formulas for building quantified achievement bullets.

## Pattern: The Master Bullet Formula

### Intent
Produce a results-first bullet from any challenge→action→result thread.

### When to Use
- Every achievement bullet — this is the default skeleton.

### Structure
```
[strong verb] + [what you did] + [quantified result] + [context / how]
```

### Example
```
Reduced [monthly shrink] by [75% / ~$1,200] by [tightening inventory controls].
 ^verb        ^what          ^quantified result        ^how
```

### Considerations
- Result before context. If the "how" is weak, drop it.
- One thread per bullet — don't chain with "and."

## Pattern: CAR-to-Bullet Compression

### Intent
Collapse a verbose CAR note into one final bullet (with an optional hook).

### Structure
```
Hook (optional):  Inherited [problem / starting state].
Bullet:           [verb] + [result vs. baseline] by [1 key action].
```

### Example
```
Hook:   Took over a center ranked 23rd of 27 with 80% client attrition.
Bullet: Improved ranking to 7th and cut attrition by half through team
        rebuild and a member-reactivation campaign.
```

## Pattern: Numbers Templates (#)

```
- Trained/Supervised [#] full-time and [#] part-time employees.
- Recruited [#] staff in [period], increasing production.
- Sold [#] products in [period], ranking [1st/2nd/3rd] of [# reps].
- Won [#] awards for [achievement].
- Exceeded goals for [#] consecutive years/quarters.
- [#] years of experience in [skill] and [skill].
```

## Pattern: Percentages Templates (%)

```
- [Skill] resulted in [%] increase/decrease in [sales/costs/clients].
- Streamlined [procedure], decreasing time on task by [%].
- Increased [customer/member] base by [%] using [skills].
- Resolved [issue], increasing customer satisfaction by [%].
- Eliminated [problem], increasing productivity by [%].
- Increased sales by [%] by [action].
- Graduated in the top [%] of class.
```

## Pattern: Dollar Templates ($)

```
- Decreased [middle-management/payroll] costs by [$] via [action].
- Averaged [$] in sales per month.
- Completed [opening/project] [$] under projected budget.
- Designed [program], earning [$] in revenue.
- Implemented [system], saving [$] daily/weekly/monthly/annually.
- Restructured [system/product] for a savings of [$].
- Collected [$] in [memberships/donations].
```

## Pattern: No Hard Metric? Quantify By...

### Intent
Add a figure when no clean sales/cost number exists.

| Quantify by | Question | Sample fill-in |
|-------------|----------|----------------|
| **Frequency** | How often? | "processed 200+ orders/week" |
| **Scale** | How many / how big? | "managed a 5,000-SKU catalog" |
| **Scope** | Across what reach? | "supported 8 sites in 3 regions" |
| **Time saved** | How much faster? | "cut a 3-day cycle to 4 hours" |
| **Rank / share** | Out of how many? | "1st of 27 centers" |
| **Before/after** | From X to Y? | "satisfaction 66% → 91%" |

## Pattern Selection Guide

| Situation | Recommended Pattern |
|-----------|--------------------|
| Writing any new bullet | Master Bullet Formula |
| Have a long CAR note to trim | CAR-to-Bullet Compression |
| Result is a count/headcount/rank | Numbers Templates (#) |
| Result is a rate or relative change | Percentages Templates (%) |
| Result is revenue/savings/budget | Dollar Templates ($) |
| No clean hard metric available | No Hard Metric? Quantify By... |
| Figure is confidential | Use %, ranges, or relative terms |
